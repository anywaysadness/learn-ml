# Основы DataFrame

***DataFrame*** - таблица с данными, где столбцы — признаки, строки — объекты  
***Серии*** -  один столбец (например, df['Age'] — серия возрастов)  


## Основные методы и значения Pandas для работы с DataFrame

- **read_csv()** читает **CSV**-файл в **DataFrame** — таблицу **pandas**  
- **head()** показывает первые **n** строк (по умолчанию 5)  
- **tail()** — показывает последние n строк
- **info()** — выводит информацию о DataFrame: типы данных, количество ненулевых значений, объём памяти
- **describe()** — показывает основные статистики (среднее, std, мин/макс, квартили) для числовых столбцов
- **isna()** создает таблицу **True/False** (True, если NaN)  
- **notna()** — противоположность **isna()**
- **median()** вычисляет медиану — значение, делящее данные пополам
- **mean()** — вычисляет среднее арифметическое (Сумма всех значений, делённая на их количество)
- **std()** — стандартное отклонение (Показывает, насколько в среднем значения отклоняются от среднего)
- **fillna(x)** заменяет **NaN** на **x**
- **dropna()** удаляет строки/столбцы с **NaN** (subset=["ColumnName"] указывает что проверяем только указанный столбец)
- **map()** применяет функцию или словарь к одному столбцу (**(df['Sex'].map({'male': 1, 'female': 0}))** здесь male → 1, female → 0)  
- **apply()** — применяет функцию по строкам или столбцам (axis=0 — столбцы, axis=1 — строки)
- **value_counts()** — считает частоту значений в столбце
- **groupby()** — группирует данные по одному или нескольким столбцам для агрегации
- **sort_values()** — сортирует строки по значению одного или нескольких столбцов
- **drop()** — удаляет указанные строки или столбцы (columns=["ColumnName"] или index=[...])
- **rename()** — переименовывает столбцы или индексы (columns={"old": "new"})
- **to_csv()** — сохраняет DataFrame в CSV-файл

- **shape** — возвращает кортеж (число_строк, число_столбцов)
- **columns** — список имён столбцов
- **dtypes** — типы данных в каждом столбце
